# Images 폴더

이 폴더에 프로필 사진과 포트폴리오 사진을 넣어주세요.

## 필요한 파일들

1. **profile.jpg** - 메인 프로필 사진 (권장 크기: 400x400px 이상, 정사각형)
2. **photo-1.jpg ~ photo-20.jpg** - 포트폴리오 사진들 (권장 크기: 800x800px 이상)

## 파일 형식

- JPG, JPEG, PNG 형식 지원
- 파일명은 위와 같이 정확히 맞춰주세요

## 참고사항

- 사진은 자동으로 최적화되어 표시됩니다
- 반응형으로 다양한 화면 크기에 맞게 조정됩니다
